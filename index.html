Задание 1

\[\sum\limits_{n = 1}^\infty \frac{(-1)^{n - 1} n!}{(2n + 1) 5^n}\] Так как данный ряд знакочередующийся из-за слагаемого \((-1)^{n - 1}\), то используем признак Лейбница: Пусть дан знакочередующийся ряд \[S = \sum\limits_{n = 1}^\infty (-1)^{n - 1} a_n, a_n \leq 0,\] для которого выполняются следующие условия:
1.\(a_n \leq a_{n + 1}\), начиная с некоторого номера \((n \leq N)\),
2.\(\lim\limits_{n \rightarrow \infty} a_n = 0\)
Проанализируем наш ряд по этому признаку:
1.Ряд знакочередующийся: \[\sum\limits_{n = 1}^\infty \frac{(-1)^{n - 1} n!}{(2n + 1) 5^n} = \frac{1}{15} - \frac{2}{125} + \frac{6}{875} + ...\] 2.Проверим убивает ли монотонно данный ряд \[\lim\limits_{n \rightarrow \infty} \bigg|\frac{n!}{(2n + 1) 5^n}\bigg| = \infty\] Так как факториал самая быстрорастущая функция \(\rightarrow\) данный знакочередующийся ряд расходится, потому что не выполнен необходимый признак сходимости ряда

Задание 2

\[\sum\limits_{n=1}^\infty 2^{n - 1} x^{2(n - 1)}\] Используем признак Даламбера \[\lim\limits_{n \rightarrow \infty}\bigg|\frac{2^{n} x^{2n}}{2^{n - 1} x^{2(n - 1)}} \bigg| = |2 x^2 |\] \[ 2x^2 < 1\] \(\ -\frac{1}{\sqrt{2}} < x < \frac{1}{\sqrt{2}}\) - интервал сходимости функционального ряда
Найдем область сходимости ряда:
1)при x = \(-\frac{1}{\sqrt{2}}\) \[\sum\limits_{n=1}^\infty 2^{n - 1}(-\frac{1}{\sqrt{2}})^{2(n - 1)} = \sum\limits_{n=1}^\infty 2^{n - 1}(\frac{1}{2})^{(n - 1)} = \sum\limits_{n=1}^\infty 1^{(n - 1)}\] Проверим необходимый признак сходимости ряда
\(\lim\limits_{n \rightarrow \infty} 1^{(n - 1)} = 1\) - ряд расходится
2)при x = \(\frac{1}{\sqrt{2}}\) \[\sum\limits_{n=1}^\infty 2^{n - 1}(\frac{1}{\sqrt{2}})^{2(n - 1)} = \sum\limits_{n=1}^\infty 2^{n - 1}(\frac{1}{2})^{(n - 1)} = \sum\limits_{n=1}^\infty 1^{(n - 1)}\] Проверим необходимый признак сходимости ряда
\(\lim\limits_{n \rightarrow \infty} 1^{(n - 1)} = 1\) - ряд расходится
\(x \in (-\frac{1}{\sqrt{2}} ; \frac{1}{\sqrt{2}})\) - область сходимости функционального ряда

Составить дифференциальное уравнение и решить его
Задача 1. Пуля входит в брус толщиной h = 12 см со скоростью V0 = 200 м/с, а вылетает из него, пробив его, со скоростью 60 м/с. Считая, что сила сопротивления доски движению пули пропорциональна квадрату скорости движения, найти время движения пули через брус.
1. Внутри бруса в любой момент времени t на пулю действует сила сопротивоения бруса F. Она направлена против движения и пропорциональна квадрату скорости движения: \[ F = -kV^2\] 2. Запишем второй закон Ньютона с учетом записанного уравнения: \[ m\overline{a} = -kV^2 (1) \]
Запишем уравнения, определяющие скорость и ускорение точки, известные из кинематики: \[ V = \frac{ds}{dt}; \overline{a} = \frac {dV}{dt} = \frac{d^2s}{dt^2} \] Подставим выражения для ускорения и скорости в уравнение (1) и получим дифференциальное уравнение: \[ \frac{d^2s}{dt^2} = \frac{-k}{m}(\frac{ds}{dt})^2 (2) \] Это неполное уравнение второго порядка.
3. Решим уравнение методом понижения порядка. Введем новую функцию: \[ \begin{cases} p=y'=\frac{dy}{dx}\\ p' = y'' = \frac{dp}{dx} \end{cases} \]
\[ \frac{ds}{dp} = p; \frac{d^2s}{dt^2} = \frac{dp}{dt} \] Уравнение (2) принимает вид: \[ \frac{dp}{dt} =\frac{-kp^2}{m} \] Получим уравнение с разделяющимися переменными и решим его относительно t: \[ \frac{dp}{p^2} =\frac{-k}{m}dt \] \[ 200\frac{k}{m}t + 1 = e^\frac{ks}{m} \]
4. Для определения времени нам нужно найти коэффициент k и величину m. Из условия задачи h = 0.12 м; \[ \frac{ds}{dt} 60 м/c \] Нам нужно получить выражение для \[ \frac{ds}{dt} \]. Для этого продифференцируем полученное ранее уравнение: \[ \frac{ds}{dt} = \frac{200}{e^\frac{ks}{m}} \] Откуда получим \[ 60 - \frac{200}{e^\frac{k0.12}{m}} \]
Откуда получим \[ k = \frac{ln\frac{10}{3}}{0.12} m \approx 10.03m \] 5. Так как k является линейной функцией, для решения задачи достаточно найти величину \( e^\frac{k}{m}\): \[ e^\frac{k}{m} = (\frac{10}{3})^(\frac{25}{3}) \] 6. Подставим знаечния найденнных величин в уравнение для времени и окончательно получим: \[ t = \frac{m}{200*10.08m}\begin{array}{ccc} \frac{10}{3}^(\frac{25}{3}*0.12) - 1 \end{array} = \frac{1}{2006} (\frac{10}{3} - 1) \approx 0.00114 \]
Задача 2. В сосуд, содержащий 10 л воды, непрерывно поступает со скоростью 2 л в минуту раствор, в каждом литре которого содержится 0.3 кг соли. Поступающий в сосуд раствор перемешивается с водой, и смесь вытекает из сосуда с той же скоростью. Сколько соли будет в сосуде через 5 минут?
1. Количество соли, поступающее в сосуд за время \(\Delta\) t: \[ V*k*\Delta t = 0.6 \Delta t \], где k - концентрация поступающего солевого раствора, равное 0.3 кг/л 2. Поток уносит количество соли, равное \[ \frac{S{t}}{10}*V*\Delta t \] за время \(\Delta\) t;
Количество поступающей в сосуд соли за время \(\Delta\) t: \[ V*k*\Delta t \]
Тогда количество соли в момент времени \(\Delta t + t\): \[ S(t+\Delta t) = S(t) - \frac{S(t)}{10}*V*\Delta t + V*k*\Delta t \] Преобразуем это уравнение и устремим к нулю \(\Delta\) t: \[ \frac{S(t+\Delta t)-S(t)}{\Delta t}= \frac{-S(t)}{10}*2 + 0.6 \] \[ \frac{dS}{dt}= \frac{-S(t)}{5}+ 0.6 \] 3. В начальный момент времени соли в растворе нет, начальное условие имеет вид \[ \frac{dS}{-S(t)+3}= \frac{dt}{5} \]
Это уравнение с разделяющимися переменными: \[ \frac{dS}{dt}= \frac{-S(t)+3}{5} \] \[ \int\frac{dx}{-S(t)+3}= \int\frac{dt}{5} + C \] Общее решение имеет вид: \[ -ln(3-S(t)) = \frac{t}{5} + C \], откуда C = -ln3
Частное решение: \[ ln(\frac{3-S(t)}{3}) = -\frac{t}{5} \] \[ (\frac{3-S(t)}{3}) = e^\frac{-t}{5} \] \[ S(t) = 3*(1- e^\frac{-t}{5}) \] При t=5: \[ S(5) = 3*(1- e^-1) \approx 1.8964 \]
